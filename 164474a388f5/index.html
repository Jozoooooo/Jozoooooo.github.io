Visual Studio 插件推荐

发表于2024-06-13|经验分享
|阅读量:

Visual Assist

插件效果：代码语法着色、智能提示、重命名等……

插件链接：Visual Assist - Visual Studio Marketplace

学习版链接：飘云阁 Visual Assist X 10.9.2502.0

请支持正版

Output enhancer

插件效果：高亮提示输出窗口信息

插件链接：Output enhancer - Visual Studio Marketplace

VSDoxyHighlighter

插件效果：高亮提示Doxygen关键词

插件链接：VSDoxyHighlighter - Visual Studio Marketplace

Doxygen Comments

插件效果：一键生成Doxygen注释

插件链接：Doxygen Comments - Visual Studio Marketplace

Comment Remover

插件效果：一键删除所有注释

插件链接：Comment Remover - Visual Studio Marketplace

WakaTime

插件效果：统计每日Coding时间并生成报告

插件链接：WakaTime - Visual Studio Marketplace
